User Multipicture
http://drupal.org/project/user_multipicture
====================================


DESCRIPTION
-----------
Enables users to have multiple pictures associated with their profile.
The user can give a title to every profile picture and define which picture is the active one for display on nodes and comments.
All uploaded pictures are displayed on the user's profile page.

REQUIREMENTS
------------
Drupal 6.x

INSTALLING
----------
1. To install the module copy the 'user_multipicture' folder to your sites/all/modules directory.

2. Read more about installing modules at http://drupal.org/node/70151


CONFIGURING AND USING
---------------------
1. Go to admin/user/settings
   Click on "Enabled" in the "Pictures" fieldset
   Choose the quantity of allowed pictures per user.


REPORTING ISSUE. REQUESTING SUPPORT. REQUESTING NEW FEATURE
-----------------------------------------------------------
1. Go to the module issue queue at http://drupal.org/project/issues/user_multipicture?status=All&categories=All
2. Click on CREATE A NEW ISSUE link.
